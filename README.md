Dies war mein erstes größeres Projekt im zweiten Semester.
Es wurde mithilfe von Java und der Libary JavaFX erstellt.


Dieses Programm stellt das Sonnensystem in einer kleinern Skala dar.
Es beeinhaltes mehrere Infos über die einzelnen Planeten, die mit mithilfe von einer Tastaur Steurung aufgerufen werden kann.
Mit dem drücken einer der angegebenen Tasten wird auch auf den Planeten gezoomt.
